# Static Site Generator buitl in Python

using 
- python
- Hugo [site](https://gohugo.io/)


## using pythons built - in web server

```cli
cd public
python3 -m http.server 8888
```

open in browser
`http://localhost:8888`

Default Base URL: http://localhost:8888
Optionally configure your CLI to override the default base URL by running bootdev configure base_url <url>
